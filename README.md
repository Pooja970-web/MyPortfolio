## ⚙ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/your-username/portfolio-react.git
```

Navigate into the folder:

```bash
cd portfolio-react
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
